# 保研计算器算法说明

本文档整理 `src/pages/tool/postgrad/index.jsx` 中的两类核心逻辑：

1. 推免成绩计算算法
2. 文本框成绩导入与课程映射算法

---

## 1. 推免成绩计算算法

### 1.1 成绩与绩点换算

- 百分制课程绩点：
  - `score < 60` -> `0`
  - `60 <= score <= 100` -> `4 - 3 * (100 - score)^2 / 1600`
- 五级制映射：
  - 优秀 `4.0`
  - 良好 `3.5`
  - 中等 `2.8`
  - 及格 `1.7`
  - 不及格 `0`
- 二级制通过/不通过用于达标判断，不参与推免成绩计算。

### 1.2 可计入规则

- 仅可计入课程参与推免成绩：
  - 非五级制
  - 非二级制
  - 已填写成绩
- 未填写成绩课程不参与推免成绩计算。

### 1.3 基础平均 GPA（baseAvgGpa）

- 从当前年级所有已填写且可计入课程中计算：
  - `baseAvgGpa = sum(课程绩点 * 学分) / sum(学分)`
- 若无有效课程，`baseAvgGpa = 0`。

### 1.4 模块计算方式

#### A. 非 selectable 模块

- 达标判断：
  - 按模块 `minCount` / `minCredits`
  - 只看“已填写成绩”的课程数量和学分
- GPA 计算：
  - 仅统计“已填写且可计入”的课程

#### B. selectable 模块

- 左侧勾选表示“是否修读”。
- 算法只从“已勾选 + 已填成绩”的课程中挑选。
- 若配置 `requiredNames`：
  - `requiredMode = score`：必修课程必须填成绩
  - `requiredMode = passfail`：必修课程必须 `passStatus = pass`
- 可选课程部分：
  - 若配置 `minCount`，按绩点从高到低取前 N 门
  - 若配置 `minCredits`，按绩点从高到低累加到目标学分
- 模块达标：
  - 必修条件满足
  - 且可选部分满足 `minCount` 或 `minCredits`

#### C. 23 届外语课（C）特殊规则

- 必修：英语阅读（1）、英语写作（1）
- 组选择：大学英语 A（1）（2）或大学英语 B（1）（2）
- 该组由用户手动选择（A/B），仅所选组参与纳入与达标。

### 1.5 总推免成绩

- 汇总所有模块纳入课程的：
  - 总学分 `totalCredits`
  - 总学分绩点和 `totalQp`
- 最终平均：
  - `avg = totalQp / totalCredits`（当 `totalCredits > 0`）

### 1.6 绿色勾选（√）

- 显示条件：
  - 课程在模块纳入集合 `included` 中
  - 且课程可计入（非五级、非二级）

## 2. 文本导入与课程映射算法

### 2.1 输入

- 用户在导入文本框中粘贴教务成绩查询响应文本。
- 支持：
  - 制表符分隔
  - 多空格分隔（兜底）

### 2.2 行解析

每行会尝试识别：

- 课程名
- 课程号（用课程号格式模式定位列）
- 学分
- 总成绩
- 替代课程名（若存在）

成绩类型识别：

- 百分制：可解析为数字
- 五级制：`优秀/良好/中等/及格/不及格`
- 二级制：`通过/不通过`

### 2.3 课程名规范化

匹配前做规范化处理：

- 中英文括号统一
- 去空格
- 小写化
- 课程名别名修正（如 `基础物理学A(1)` -> `基础物理学(1)`）

同时构建“松散键”用于兜底匹配（只在唯一命中时采用，防止误匹配）。

### 2.4 映射策略

1. 优先精确规范化键匹配
2. 精确失败时使用松散键唯一匹配
3. 命中后写入对应课程行
4. 若当前行含“替代课程名”，会同时尝试将该行成绩写入替代课程名对应课程（用于替代修读场景）
5. 若命中课程所在模块为 selectable，则自动勾选该课程

写入规则：

- 百分制：填 `score`
- 五级制：仅在目标课程本身是五级制时写入对应选项值
- 二级制：写 `passStatus`（`pass` / `fail`）
- 学分若可解析且大于 0，同步覆盖课程学分
- 当“替代课程名”命中保研所需课程时，使用当前行成绩进行替代填充



### 2.5 导入反馈

- 文案展示：
  - `已识别 X 条`
  - 若存在未匹配，附加“未匹配示例（最多 5 条）”

---

### 2.5 大英 A/B 自动识别

- 导入后会扫描文本中的外语课程命中情况：
  - `大学英语A（1）（2）`
  - `大学英语B（1）（2）`
- 若检测到命中，会自动设置外语课分组选择（A 或 B）。
- 若 A/B 同时出现，按命中数量较多的一组自动选择。

## 3. 缓存与迁移

- 成绩数据缓存：`tool:postgrad:data:v10`
- 分组选择缓存：`tool:postgrad:groups:v1`
- 页面加载时执行数据规范化与迁移（课程名、分组结构、必修课程 scoreType 等）。
