# 🐧 Linux 机使用与配置手册

---

## 🛠 拥有属于你的 Linux 机

目前主流的三种途径：

### 1. 云服务器 (VPS)

- **特点**：使用 DigitalOcean、Azure 等（GitHub Student Pack 可免费）。**最贴合实际开发场景**。
- **难度**：**中**
- **教程**：[云服务器配置实战](https://www.bilibili.com/video/BV1aW421w7ZS/)

---

### 2. 虚拟机 (VMware/VirtualBox)

- **特点**：在现有系统内模拟运行。**适合初学者练手，不慎搞坏可秒恢复**。
- **难度**：**低**
- **教程**：[虚拟机安装教程](https://www.bilibili.com/video/BV1M94y1U7nc/)

---

### 3. 物理机安装 (原生系统)

- **特点**：通过 U 盘/ROM 制作启动盘。性能最强，但**涉及 BIOS 调节，过程复杂**。
- **难度**：**高**
- **教程**：[物理机双系统安装](https://www.bilibili.com/video/BV1554y1n7zv/)

---
