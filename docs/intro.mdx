# 资料介绍

欢迎来到 **Soft Where** 资源中心。这里是为软件学院学子量身打造的一站式资源枢纽，旨在汇聚各类优质资料，消除信息差，陪伴大家在软院的学习与生活。

通过本页面的动态导航，你可以快速纵览“资源”板块下的所有功能。无论是课程指南、科研利器，还是两校区的校园地图，我们都已为你整理归类，点击卡片即可直达。

---

### 🚀 探索指南

为了让你更高效地获取所需，我们建议从以下维度开始探索：

- **学术导航**：在 [培养方案](/docs/category/培养方案) 中明确学业要求，从 [常用指令](/docs/category/常用指令) 中掌握开发基本功。
- **科研助力**：前往 [科研工具](/docs/科研工具) 获取前沿论文检索建议与高效工具集。
- **校园生活**：通过 [校园地图](/docs/category/校园地图) 熟悉环境，在 [北航校历](/docs/北航校历) 中掌握学期节奏。
- **开源共建**：如果你发现有价值的资源尚未收录，欢迎查阅 [贡献指南](/docs/贡献指南)，与我们一同完善这片属于软院人的知识森林。
