# 贡献指南

欢迎参与软件学院(21系)学生会官网共建。这里提供清晰的贡献入口和提交规范，后续也会持续扩展更多开源与共享渠道。

## 贡献方式总览

- 开源仓库: 提交 fork 申请与合并请求
- 资料与文件: 向网盘或资料库提交文件

## 开源仓库贡献

如果你希望参与开源仓库共建, 可通过填写以下问卷提交申请, 后续通过审核后会被 fork 进入学生会官方仓库:

- 入口: https://jsj.top/f/cYWiRt
